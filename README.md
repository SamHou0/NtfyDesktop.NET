# NtfyDesktop.NET

Yet another [Ntfy](https://github.com/binwiederhier/ntfy) cross-platform GUI client, built with .NET and Avalonia.

## Features

- Websocket connection to ntfy server
- Real time message receiving
- Easy-To-Use GUI with Material Design
- Native notification via `notify-send` on Linux Desktops
- Auth token to access your server
- Cross platform support
  - Linux
  - Windows (Work in Progress!)
